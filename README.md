# Portfolio

A clean and modern personal portfolio website built with HTML, CSS, and JavaScript.

## Structure
- index.html — main page
- css/styles.css — styling
- js/app.js — interactions and animations
- assets/images — project and profile images
- assets/fonts — custom fonts

## Preview locally
Open index.html in a browser, or serve the project with a simple static server.

## Deploy to GitHub Pages
1. Push this repository to GitHub.
2. Open the repository settings.
3. Enable GitHub Pages and select the root folder.
